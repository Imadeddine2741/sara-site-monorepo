# Sara Backend API
API REST pour le site de Sara Rahmani - Psychologue clinicienne

## Fonctionnalités
- Prise de rendez-vous en ligne
- Génération automatique factures PDF
- Gestion webinaires/conférences
- Blog neurosciences
- Authentification JWT sécurisée
- RGPD compliant

## Stack
Spring Boot 3.2+ | MariaDB | JWT | PDF Generation
